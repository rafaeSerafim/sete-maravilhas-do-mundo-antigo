# Style Guide - Sete Maravilhas do Mundo Antigo

##Tecnologias
* HTML
* CSS puro

##Bootstrap
Será utilizado o bootstrap para:
* Layout
* Grid
* Botões
* Resposividade

##CSS
Será utilizado o CSS para:
* Cores
* Estlização
* Ajustes visuais

###Nomes de classes
Nomes em inglês e descritivos
Exemplo: .navbar, .gallery, .card, .title

###Padrão CSS
Utilizar padrão BEM (Block Element Modifier)
Exemplo:
.card
.card__image

###Cores
Utilizar hexadecimal (#)

##Root
:root {
  --color-title: #D85C1F;
  --color-text: #F2F2F2;
  --color-navbar: #1B3659;
  --color-background: #ACEBF2;
}
